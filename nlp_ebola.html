NLP for Classifying Tweets About Ebola

This project is the final assignment for LHS 712: Natural Language Processing for Health, a graduate-level course on the application of natural language processing tools to health data. I decided to build a classifier to categorize tweets posted during the 2014 Ebola outbreak in Western Africa. Twitter has been used extensively for natural language processing and I was interested in exploring Twitter NLP for use in emergency health situations.
